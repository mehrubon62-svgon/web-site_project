# About the Project

## Why this project was created
This project was created as a full-featured e-commerce platform for computer hardware and PC components using Django.

Main goals:
- simplify the process of choosing and buying PC components;
- provide a convenient product catalog, shopping cart, and checkout flow;
- add intelligent and personalized features (PC configurator, AI consultant, reviews, and user profile).

## What is implemented
- Product catalog (processors, GPUs, motherboards, RAM, storage, and more).
- Detailed product pages with images and specifications.
- User registration, login, and profile management.
- Shopping cart, order checkout, and order history.
- PC configurator for building compatible setups.
- AI consultant to help users choose products.
- Reviews and review replies.
- Wishlist and additional user-focused features.

## Project advantages
- Modular Django architecture (`orders`, `reviews`, `configurator`, `ai_consultant`, etc.) that is easy to extend.
- Flexibility: new product categories, filters, and business logic can be added quickly.
- Scalability: suitable for both learning purposes and MVP launch.
- Better user experience: a clear flow from browsing to purchase.
- Practical value: demonstrates a full e-commerce development cycle.

## Who this project is for
- Users who want to build and buy a PC quickly.
- Developers who need a solid foundation for extending an online store.
- Portfolio use to showcase full-stack web development skills.

## Summary
This project was created to combine a hardware catalog, compatible PC build configuration, and a seamless purchasing process in one web application.
